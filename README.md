# Final-Project

This project allows for groups to most efficiently handle shared expenses over a long period of time. The project is targeted specifically toward college friend groups / roommates to track expenses over a semester or academic year. The program will take the expenses of all individuals in the group over time and on whose behalf they were made, and once prompted, will calculate the smallest number of transactions necessary to correctly redistribute what is properly owed. It will produce a comprehnsive report, not only being the necessary transactions to settle debts, but also the name of the individual who spent the most money, who paid the most amount of times, and the description of the most expensive expense. My hope is that this helps users pursue more effective spending.
